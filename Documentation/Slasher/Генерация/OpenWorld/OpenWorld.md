Основной мир представляет собой бесконечную карту, которая генерируется чанками по мере исследования игроком. Каждый чанк является независимой игровой единицей и содержит собственное наполнение.

Каждый новый чанк сначала определяется как принадлежащий определённому биому, после чего наполняется содержимым согласно правилам этого биома.

FROM [[]]
SORT file.name ASC
